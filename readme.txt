== Description ==

Universal Post View Counter is a lightweight and free WordPress post views and live visitor counter plugin that tracks and displays total post views and real-time active users on posts, pages, and custom post types. It works seamlessly with JetEngine, ACF, WooCommerce, Toolset, Metabox, Pods, and all WordPress page builders like Gutenberg, Elementor, Divi, Oxygen, Brizy, Kadence Blocks, and more. Compatible with all WordPress themes, both classic and Full Site Editor (FSE) themes, this plugin offers an AJAX-based, fast, and privacy-friendly way to monitor your website post statistics with no third-party tracking needed.










How to Use?  

1?. Install & activate the plugin.  
2?. Add `[post_views]` to display total views.  
3?. Add `[live_viewers]` to show real-time visitor count.  
4?. Customize settings from the WordPress dashboard.
